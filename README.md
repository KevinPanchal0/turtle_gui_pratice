# Turtle Graphics Art 🎨

A Python project that uses the **turtle graphics library** to create different geometric patterns and visual designs.

The program includes multiple drawing functions such as squares, dashed lines, polygons, random walks, and spirograph-style art.

---

## Features

- Draw a square
- Create dashed lines
- Draw dashed squares
- Generate polygons with different sides
- Random walk pattern
- Random RGB colors
- Spirograph pattern

---

## Project Overview

This project demonstrates:

- Python functions
- Loops
- Random number generation
- Turtle graphics
- Basic geometry concepts
